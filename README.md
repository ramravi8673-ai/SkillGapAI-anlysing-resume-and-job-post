SkillGapAI is an AI-powered system that analyzes resume and job description skills
to identify skill gaps using NLP and semantic similarity.

## Features
- Resume and Job Description skill extraction
- Semantic similarity using Sentence-BERT
- Skill gap classification (Matched, Partial, Missing)
- Radar chart and heatmap visualization
- CSV report export
- Streamlit-based web interface

## Technologies Used
- Python
- Streamlit
- Sentence-BERT
- Scikit-learn
- Matplotlib
- Pandas
